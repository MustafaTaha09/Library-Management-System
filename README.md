# Library-Management-System
A Library System which handles all kinds of features from an admin&amp;user POV designed based on the OOP principles.
## Features
for an Admin:
- Add book
- View books
- Search/Delete a book
- View orders
- Delete all data

for a User:
- Borrow a book
- Return a book
- Place order
- Search

## OOP Principles Used
- Inheritance
- Polymorphism
- Abstraction
- Interfaces
